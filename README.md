# approval-testing-app

Approval testing app is a basic web application that has a couple of car details stored inside it that you can access via a web page (http://localhost:8080) and toggle between by pressing the toggle button on the web page.

It was primarily created as an application under test for [Approval-testing-koans](https://github.com/mwinteringham/approval-testing-koans) to help people practise approval testing techniques.

## Getting setup

To run it, simply download the latest JAR [from the releases page](https://github.com/mwinteringham/approval-testing-app/releases) and then run ```java -jar approval-testing-app-x.x.jar``` to get it started. From there you will be able to access it's webpage on [http://localhost:8080/](http://localhost:8080)
